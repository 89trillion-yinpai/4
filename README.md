# 排行榜技术文档
## 整体框架：
- 1:先通过提供的json文件读取数据并创建对象，在创建排行榜的时候解析这些数据成为他们的属性。
- 2:通过json给的数据判断需要创建的预置物。
- 3:导入美术资源
- 4:根据需求文档确定整个UI的层级和结构。
- 5:根据需求文档实现具体的功能并测试
